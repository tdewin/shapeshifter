# shapeshifter
Google slide SVG importer for Text and Rectangles. Tuned for pikchr boxes and text outputs. It will try to detect boxes from path (doesn't work with rad).

Recommended workflow: create import slidedeck that empty with plugin and blank slides. Import in this deck, resize and copy paste to your target deck the item.

Heavily Vibe Coded plugin. Not meant to support the full SVG spec (impossible due to Googles limited API)
